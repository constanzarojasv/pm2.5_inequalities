# Spatiotemporal Analysis of PM2.5 Exposure in the Santiago Metropolitan Region (2022-2024)

This repository contains the code and data structure used to analyze the spatiotemporal trajectories of fine particulate matter (PM2.5) pollution across 10 municipalities in the Santiago Metropolitan Region, Chile. 

The project evaluates the impact of sociodemographic variables (multidimensional poverty, population density) and seasonal factors (winter-biomass interaction) on PM2.5 concentrations, contrasting national regulatory compliance with US Environmental Protection Agency (EPA) and World Health Organization (WHO) guidelines.

## 📂 Repository Structure

The project follows a structured workflow to ensure the reproducibility of the analysis:

* **`input/`**: Contains raw and processed data.
    * `data_raw/`: Original `.csv` files with hourly PM2.5 records per monitoring station (SINCA).
    * `data_processed/`: Cleaned and consolidated `.rds` dataset.
    * `shp_SINCA/`: Vector files (shapefiles) of the monitoring stations.
* **`output/`**: Results generated by the scripts.
    * `tables/`: Descriptive statistics, healthy days tables, and mixed models results (in `.csv` and `.md` formats).
    * `figures/`: High-resolution spatial maps (`.png`) of concentrations, 98th percentile, and regulatory compliance.
* **`processing/`**: R scripts that make up the main analysis pipeline (see the "Workflow" section).
* **`reports/`**: This folder will contain the final manuscript and its corresponding DOI once published. *(Pending upload)*

## ⚙️ Workflow (Pipeline)

The analysis is divided into 5 sequential scripts located in the `processing/` folder:

1.  **`00_setup_functions.R`**: Loads libraries, defines spatial color palettes, manual sociodemographic data (CASEN 2022 / Census 2024), and automated functions for daily data cleaning.
2.  **`01_load_clean.R`**: Reads and standardizes the 10 raw databases, calculates daily averages, applies temporal filtering (2022-2024), and consolidates a master dataset for analysis.
3.  **`02_descriptive_analysis.R` & `02b_healthy_days_table.R`**: Generates annual and winter descriptive statistics (ANOVAs) and calculates the proportion of "Healthy Days" according to the regulatory limits of Chile (50 µg/m³), the US EPA (35 µg/m³), and the WHO (15 µg/m³).
4.  **`03_mixed_models.R`**: Fits Linear Mixed Models (LMM). Runs a model with interaction effects (Winter × Biomass) and a fully adjusted model controlling for density and poverty to identify the drivers of pollution peaks.
5.  **`04_spatial_analysis_maps.R`**: Spatial integration using the elevation API and the `chilemapas` package. Generates the study area cartography, visual interpolation of regulatory compliance, and the geographic distribution of averages and the 98th percentile.

## 🛠️ Technical Requirements

The analysis was developed in **R**. To replicate the environment, the following main libraries are required:

* **Data Manipulation:** `dplyr`, `tidyr`, `lubridate`, `readr`, `janitor`, `purrr`.
* **Statistical Modeling:** `lme4`, `lmerTest`, `broom.mixed`, `performance`.
* **Spatial Analysis & Visualization:** `sf`, `chilemapas`, `ggplot2`, `elevatr`, `ggspatial`, `shadowtext`.
